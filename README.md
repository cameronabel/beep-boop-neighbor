# TDD Elements

## Describe:  numAnalyzer()

#### Test: It should return a string containing the number that was provided.
**Code:**\
const num = 2;\
numAnalyzer(num);\
**Expected Output: "2"**

#### Test: When passed a 3, it should return "Won't you be my neighbor?".
**Code:**\
const num = 3;\
numAnalyzer(num);\
**Expected Output: "Won't you be my neighbor?"**
